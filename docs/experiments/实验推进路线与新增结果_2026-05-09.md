# 实验推进路线与新增结果 2026-05-09

## 结论先行

本轮应把论文实验主线收窄到 **connectivity-first / grammar-native connected scaffold / per-depth projection**。已有证据可以支持的 claim 是：

1. 对选定的 vine、bismuth-like、pyrite-like、porous/coral-like case，grammar-native scaffold 能在同一 case 的 depth 或 density 参数变化下保持 occupancy 6-neighborhood 连通。
2. Trellis2 texture/PBR export 可作为 selected projected meshes 的资产兼容性证据，但不能单独证明拓扑。
3. hard-DLA bridge 和 post-hoc repair 只能作为 negative/control：有时 face-level component 变好，但 occupancy support 或视觉可信度仍不足。

不能强写的 claim 是：

- 不能说已经系统性优于所有 traditional / mesh-space / direct sparse baselines。
- 不能说 DLA/frontier physical growth 已解决。
- 不能说 mesh-face topology 已完全 clean。
- 不能把 texture/PBR 作为本文核心贡献。

## Case 选择闭环

推荐主文候选：

1. **Vine/root/tree**：用于递归 depth 与 projection-stabilized execution 主线，但需要 neutral junction/root/tip zoom。
2. **Bismuth-like / pyrite-like connected scaffold**：用于 non-tree transform-copy / lattice / crystal-like 视觉线，但 caption 必须避免物理晶体生长 claim。
3. **Coral density parameter sequence**：用于同一 grammar 参数控制，当前是最稳的非树 method-behavior 图。

推荐 appendix/control：

1. hard-DLA bridge ablation。
2. fork variants where per-depth projection still fails。
3. raw face-component fragmentation diagnostics。

不建议继续投入：

- 把 hard-DLA bridge 当主文正例继续 texture/rerender。
- 只为了 gallery 扩展更多 PBR 图。
- 没有同一 case metric 闭环的单张漂亮图。
