## 总体结论

当前证据可以支撑的主线很窄：**per-depth projection/admissible-state selection 在若干 conservative case 中抑制 fragment propagation；selected projected/scaffold meshes 可进入 Trellis2 texture/PBR export path；传统 L-system / space-colonization 在有利 tube-occupancy 条件下也很强，因此不能把主 claim 写成“连通性击败传统 baseline”。**

不能支撑的强 claim：sparse-latent recursive language 已严格优于 mesh-space procedural postprocessing；masked naturalization 已稳定提升质量且不漂移；effective resolution 超过 one-shot generator；DLA/physical crystal growth 已解决；texture/GLB face topology clean。

## P0 缺失 / 未闭合 Top 5

1. **latent-space vs mesh-space recursion 正式对比缺失**：现有 texture-only 诊断不足以证明 generation-model-native recursive language 的必要性。
2. **完整 same-root projection-stability matrix 缺失**：tree/root/vine 还没有 traditional、direct sparse、final-only、prune-per-depth、model-bridge、traditional repair、proposed 的统一矩阵。
3. **masked naturalization ablation 缺失**：Flow/SDE sampler-only 是 negative/control，尚不能证明 masked local naturalization 提升表面质量且不造成 topology drift。
4. **effective resolution / recursive refinement 缺失**：runtime/token profiling 不能替代 one-shot vs recursive refinement 与 local detail preservation 证据。
5. **fixed-camera zoom QA 缺失**：没有 root/junction/tip/failure zoom，scalar connectivity 容易掩盖 fake bridge、surface soup、endpoint breakage。
