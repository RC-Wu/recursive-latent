## 0. 总体结论

当前项目已经从“递归 Trellis2 工程管线”推进到较清晰的论文主线：**Projection-Stabilized Recursive Sparse-Latent Grammar / PS-SLG / PS-RSLG**。最稳的核心不是“解决所有递归 3D 生成”，而是：

> 在冻结 3D 生成模型的 sparse latent / sparse support 状态上执行有限深度递归语法；语法负责 topology、anchors、rule proposal、competition 和 projection；Trellis2 负责 sparse state 接口、decode/re-encode、局部 naturalization 和 selected texture/PBR export。关键证据是 per-depth projection 对 voxelized 6-neighborhood connected support 的稳定作用。

必须保守写：

- 可以主张：有限深度 recursive asset growth、per-depth projection 抑制 fragment 传播、selected projected scaffold 可进入 Trellis2 texture/PBR export。
- 不能主张：raw GLB mesh topology clean、真实 DLA/晶体物理生长、无限递归已解决、texture/PBR 是拓扑证明、matplotlib/point cloud 图可作正式可视化。
- DLA bridge / post-hoc bridge 是负结果；Trellis2 texture/PBR 是资产化兼容能力，不是 topology proof；matplotlib scatter / point-cloud preview 只能作内部诊断。

## 16. 当前最重要的 10 个缺口

1. **公平 baseline 闭环不足**：缺同 root、同 depth、同 seed、同 renderer 下的 L-system / space-colonization / direct / final-only / prune-per-depth / bridge-per-depth / proposed 完整矩阵。
2. **Branch/topology 语义指标不足**：root reachability、orphan tips、junction、tip count、branch length/angle、path span 仍不系统。
3. **Occupancy connectedness 与 raw mesh topology 的边界必须继续公开**：不能让 GLB face fragmentation 被 texture render 掩盖。
4. **DLA 仍未解决**：post-hoc bridge 是负结果；缺真实 frontier / porosity / cavity / fake-bridge 指标。
5. **Effective resolution 缺闭环**：缺 one-shot vs recursive refinement、token budget、local detail density、zoom-level detail retention。
6. **Cache/LOD/infinite 仍是 proxy**：远端 run 失败，缺 true Trellis decode/texture 验证；不能主张无限递归。
7. **Naturalization ablation 不完整**：rule-only、no naturalization、weak blend、masked local、global flow、with/without projection 未统一对比。
8. **Texture/PBR 诊断表仍需规范化**：GLB/PBR channels、UV seams、warnings、render pass/fail、material continuity 需完整报告。
9. **主图视觉仍需论文级重排**：白底、无边框、无 PPT 标签、固定相机、zoom crop、caption 一图一 claim。
10. **论文 claim 仍需压缩**：DLA/crystal/infinite/material recursion 等过强表述必须降级或移到 supplement/limitations。

## 18. 最终判断

项目目前最接近可投稿的版本是一个诚实、收窄的 SIGGRAPH Asia 方法论文：

> 我们提出 projection-stabilized recursive sparse-latent grammar；它在冻结 3D generator 的原生 sparse state 上执行有限深度递归程序，用 per-depth projection 把 connected support 变成递归状态不变量。实验最强地支持 vine/tree conservative competition 和 selected non-tree connected scaffold；Trellis2 texture/PBR 证明资产化兼容性，但不证明拓扑正确；DLA bridge、cache/LOD、infinite recursion 仍是边界或未来工作。

按这个边界写，当前证据链可以继续推进；若写成通用拓扑干净的 textured recursive asset generator、真实 DLA/晶体生长、或无限递归系统，证据会被 reviewer 直接击穿。
